CSS 포지션 속성

 
캔버스는 canvas태그로 작성
캔버스는 html이 보여지는 모든 환경에서 동일하게 보여진다.
모바일에서 보임.
캔버스를 통해 선,원,사각형,곡선,이미지 등을 그릴 수 있고
이미지 합성 및 변환이 가능하다.

기본작성 방법:

	
캔버스 출력되지 않으면 나오는P



1.캔버스를 만들면 캔버스를 객체화 해야한다.
2.캐너스 컨텍스트(그림그리는 도구:렌더링 툴)를 얻는다. getContext() 메소드 활용
3.캔버스 컨텍스트 객체의 메서드를 이용하여 그림을 그린다.

canvas 객체의 프로퍼티와 메서드
프로퍼티: width, height, style 높이 너비 스타일시트 속성
메서드 : getContext() 캔버스 그림을 그리는 컨텍스트 객체를 리턴한다.

context객체의 프로퍼티와 메서드.
프로퍼티
1.strokeStyle : 선의 색
2.fillStyle: 채우기 색
3.lineWidth: 선 두께(픽셀단위)
4.font: 텍스트 폰트 설정.
5.textAlign 텍스트 정렬 방식.

메서드
1.beginPath() : 이전 경로를 모두 지우고 새로 시작
2.closePath(): 경로의 끝 점과 시작점을 연결하는 선을 추가하고 경로를 닫는다.
3.moveTo() 경로에 새 점을 만든다.
4.lineTo(): 경로의 끝 점과 연결하는 선을 추가
5.rect(): 경로에 사격형 추가
6.arc(): 경로에 원 추가
7.fill() :  경로에 그려진 모든 도형을 외곽선 없이 내부에 색을 채워 그린다.
8.stroke() : 경로에 그려진 모든 도형을 외곽선만 그린다.
9.clearRect() : 캔버스에서 사각형 영역을 지운다.

10.fillRect() : 캔버스에 외곽선 없이 사각형 영역을 색으로 채워 칠한다.  //#경로 abcd 변이있음
11.strokeRect() : 사각형을 경로에 추가하지 않고 바로 캔버스에 그린다.
 //한번에 사각형이틀에 찍히는것맹키로 생김 beginPath가 없다.

12. strokeText() : 텍스트를 캔버스에 바로 그린다.
13.drawImage(): 이미지를 캔버스에 바로 그린다.

#도형을 그리는 과정
직선,사각형, 원호등의 도형은 텍스트나 이미지와 다르게 2단계를 거쳐 캔버스에 그려진다.
1.Path(경로)만들기
2.캔버스에 경로에 담긴 도형을 모두 그리는 과정

경로란
캔버스 위에 펜이 지나가는 길
moveTo() lineTo() rect() arc() 등 메서드는 경로를 추가하는 메서드
경로를 전부 추가하고 그 후에 stroke() or fill() 로 캔버스에 그린다.
