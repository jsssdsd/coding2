#
CSS3스타일 적용방법
1. 헤드 부분에 style태그 내부에 스타일 시트 작성
2.HTML 태그 속성으로 스타일 시트 적용	
3. 외부 스타일 시트 파일을 따로 생성하는 방법 link나 import로 불러옴

#
외부스타일 시트 만드는 방법
확장자가 .css인 파일을 만든다
.css파일 내부에 스타일 시트 내용 작성
불러오는 방식 2가지 ( link, @import )

#
CSS3스타일 시트의 상속 
HTML구조에서 부모(parent element)에게 적용된 스타일 시트는 
자식요소에게 전부 동일하게 적용이 된다.
HTML페이지 전체에 적용하고 싶으면 body에 적용하면 된다.


#
1. 모든 태그마다 브라우저에서 기본적으로 적용되는 스타일이 존재
2. .css 외부 스타일 시트 파일
3. HTML 내부 스타일 시트
4. 태그의 style속성에 작성된 스타일

4가지  스타일 요소 사이의 우선순위.
오버라이딩은 덮어쓰기로 동일한 CSS3 프로퍼티에 각 다른값이 
설정되면 우선수위가 높은 스타일을 적용하게 된다.

#
셀렉터 CSS3스타일을 적용할 대상물
id: 고유한 이름(중복x)
class: 중복가능한이름
태그명: HTML 태그명 


1)태그이름 셀렉터
태그명 자체가 셀렉터가 되는 방식
같은태그로 구성된 모든 요소에 스타일이 반영된다.

2)class이름 셀렉터
셀렉터이름 앞에 . 을 쓴다.
HTML태그의 class명 속성으로 지정하는 방식
태그명. 클래스명 속성으로 지정하는 방식
해당 클래스명을 가진요소에만 적용가능
셀렉터는 조합이 가능하다.

3) ID이름 셀렉터
셀렉터 이름앞에 #을 쓴다.
HTML태그의 ID명 속성으로 지정하는 방식
HTML 태그의 ID명 속성은 고유하게 설정한다.(중복 불가)

4)전체 셀렉터*
셀렉터 이름 없이 *을 쓴다
문서에 포함되어 있는 모든 HTML요소에 공통으로 적용한다.

5)type셀렉터
HTML 태그의 특정 속성에 대해 일치하는 태그에만 스타일을 적용하는 방식

6) 조합 셀렉터

자식셀렉터 '>'
자식 셀렉터는 부모 자식 관계인 두 셀렉터를 
>기호로 조합한다.

자손 셀렉터'' 
공백으로 자손 셀렉터 조합이 가능하다.
2세대 이상 자식 관계는 자손 셀렉터 사용한다.
!블록형태 div 블록형태 div
