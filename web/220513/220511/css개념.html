CSS3스타일 적용방법

#

#
CSS3스타일 시트의 상속 
HTML구조에서 부모(parent element)에게 적용된 스타일 시트는 
자식요소에게 전부 동일하게 적용이 된다.
HTML페이지 전체에 적용하고 싶으면 body에 적용하면 된다.


#
1. 모든 태그마다 브라우저에서 기본적으로 적용되는 스타일이 존재
2. .css 외부 스타일 시트 파일
3. HTML 내부 스타일 시트
4. 태그의 style속성에 작성된 스타일

4가지  스타일 요소 사이의 우선순위.
오버라이딩은 덮어쓰기로 동일한 CSS3 프로퍼티에 각 다른값이 
설정되면 우선수위가 높은 스타일을 적용하게 된다.

#
셀렉터 CSS3스타일을 적용할 대상물
id: 고유한 이름(중복x)
class: 중복가능한이름
태그명: HTML 태그명 






1)태그이름 셀렉터
태그명 자체가 셀렉터가 되는 방식
같은태그로 구성된 모든 요소에 스타일이 반영된다.

2)class이름 셀렉터
셀렉터이름 앞에 . 을 쓴다.
HTML태그의 class명 속성으로 지정하는 방식
태그명. 클래스명 속성으로 지정하는 방식
해당 클래스명을 가진요소에만 적용가능
셀렉터는 조합이 가능하다.

3) ID이름 셀렉터
셀렉터 이름앞에 ####한개 매우중요을 쓴다.
HTML태그의 ID명 속성으로 지정하는 방식
HTML 태그의 ID명 속성은 고유하게 설정한다.(중복 불가)

4)전체 셀렉터*
셀렉터 이름 없이 *을 쓴다
문서에 포함되어 있는 모든 HTML요소에 공통으로 적용한다.

5)type셀렉터
HTML 태그의 특정 속성에 대해 일치하는 태그에만 스타일을 적용하는 방식

6) 조합 셀렉터

자식셀렉터 '>'
자식 셀렉터는 부모 자식 관계인 두 셀렉터를 
>기호로 조합한다.

자손 셀렉터'' 
공백으로 자손 셀렉터 조합이 가능하다.
2세대 이상 자식 관계는 자손 셀렉터 사용한다. 

7)가상 셀렉터
가상 셀렉터는 특정 상황에서 적용되는 셀렉터 
사용방법: 대상물: hover

#색상
CSS에서 색상 표현 방법
1.RGB : 레드 그린 블루 3가지 값을 10진수로 0~255
	rgb(0,0,255)
2.HEX : 레드 그린 블루 16진수 -> 00~ FF
	#FF00FF
3.HSL : 색조 채도 명도 순서, 색조 0~ 360 채도와 명도 0~100%
	hsl(360,100%,100%)
4.색상명: red

CSS에서 하나의 색은 총 24비트로 표현된다.
28(0~255) 28(0~255) 28(0~255)

#box 모델
html요소를 레이아웃에서 하나의 박스형태로 나타내는것
박스의 크기 , 박스의 배경색, 박스의 여백, 테두리등

박스의 구성 요소 
컨텐츠<패딩<보더<마진
컨텐츠; HTML태그에 포함된 내용
패딩: HTML태그 컨테츠를 둘러싸고 있는 내두의 여백
보더(테두리):패딩 외부 외곽선
마진(외부 여백): 박스의 가장 마지막 영역, 테두리의 바깥 공간

패딩 : 대상물의 배경색이 그대로적용
보더: 색상을 사용자가 직접 지정
마진: 투명하게 처리되어 부모 태그의 배경색이 비춰 보인다.
