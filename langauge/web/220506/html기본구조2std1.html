#블록태그와 인라인 태그
블록 태그 예: div, h1, p, ul
인라인 태그 예: strong, a, img, span(기능없음 가장많이씀)

블록 태그 : 항상 새 라인에서 시작하고 브라우저의 왼쪽끝부터 오른쪽 끝까지 범위
인라인 태그: 해당 태그로 묶은 부분만 적용된다.

프레임은 블록 태그에 적용된다.

#메타 데이터

메타데이터 head태크안에
base/link/script/style/title/meta
base: 기본 경로 설정 URL http www// index.html


#이미지 태그 img
img태그를 사용하면 HTML 문서에 이미지를 삽입할 수 있다.
BMP,GIF,JPG,JPEG,PNG, 웹사이트의 이미지 주소 사용 가능.

이미지 태그의 속성
1.src속성: 이미지 파일의 URL
2.alt속성: 대체 문자열
3.width속성: 이미지가 출력되는 너비 (생략하면 원본 크기)
4.height속성: 이미지가 출력되는 높이(생략하면 원본 크기)

#HTML 스타일(style)
HTML 요소의 style속성을 이용하면 CSS스타일을 HTML 요소에 직접 설정 가능
형태: 태그이름 style="속성이름: 속성값"
예를 들어 배경색을 변경한다면
style="background-color:red"//배경색을 레드로
style="color:white"//폰트의 색을 화이트
style= "font-size:200%" //폰트사이즈를 200%
style= "text-align:center"//텍스트 가운데 정렬


style="background-color:red;color:white;font-size:1003%;text-align:center" 한꺼번에 다적용!!

#하이퍼링크 a태그

HTML페이지에서 다른HTML페이지로 연결되는 것: 링크
이미지, 버튼, 텍스트에 연결한다.

target속성
_self: 디폴트 값, 현재 창에서 해당 주소 문서 창이 열린다.
_blank: 현재 창이 아니라 새 창에 문서를 연다.
_parent: 현재 보다 상위 프레임에서 문서를 연다.
_top : Root 프레임에서 해당 주소문서를 연다.

#링크의 상태
link상태 : 한번도 방문한적 없는 상태(디폴트) - 파란색
visited상태: 한번이라도 방문한 상태 - 보라색
hover상태: 링크 위에 마우스를 올려놓은 상태 - 빨간색
active상태 : 링크를 마우스로 누른 상태 - CSS로 사용자가 정의

CSS3를 통해 밑줄 삭제 가능 색상을 변경 가능
