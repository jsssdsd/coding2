#전역 변수
변수의 사용법위에 따라 3가지로 나뉜다.
1. 전역 변수 : 함수 밖에서 선어된것
프로그램이 종료될때까지 메모리에 남아있는다.
2. 지역변수 : 함수내에서 선언된것
함수가 호출되면 생성
함수가 종료되면 소멸

3. 블록변수 : let 키워드로 if while for 등 블록 내에서 선언된 변수
블록 내에서만 사용가능
밖에서 접근 불가 


#let의 특징

let 키워드는 var를 사용할 때 개발자들의 실수를 줄이기 위해
var은 동일한 변수를 재선언할 수 있다.
let은 동일한 변수를 재선언X
let은 사용범위가 블록 내로 제한

var x=10
var x=20

let x=10
let x=20(x)변수 x에 대한 재선언

var로 재선언하면 기존 선언된 변수를 폐기하고 재선언

#let은 블록 변수 선언
블록 내에서만 유효한 변수로 선언시 let사용
전역에서 블록 내의 let변수 호출 분가


# 상수: 변하지않는 값을 가지는 이름 const 키워드를 사용한다.
const x =10; // 값이 선언된 후에 변경 불가능

x=20// 불가능
x=10// 불가능

const로 선언된 상수는 let과 마찬가지로 선언된 블록 범위내에서만 
사용한다.
