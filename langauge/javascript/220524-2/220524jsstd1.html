#switch 문
여러 경우에 따라 다른 코드를 실행하고 싶을 때
switch(식)
{
	case 값1:
		수행문;
		break;
	case 값2:
		수행문;
		break;
	case 값3:
		수행문;
		break;
}


모든 case에 수행문과 break를 넣어야함
switch 괄호 안에 있는 식의 결과가
case 뒤에 값과 일치하는 실행 문장으로 진입한다.
case안에 실행문 실행 후 break를 읽고 switch를 탈출한다.
case 중에서 일치하는 값이 없으면 default로 진입.

#case의 값으로 배치될 수 있는 값 종류
case1
case 1.5
case "hello"
case true
case 2+3 (=5) case 5와 동일




사용할 수 없는 값의 종류

case x 변수 사용 불가
case x >10 조건식 사용 불가
