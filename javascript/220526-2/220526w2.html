스톱워치 

1.페이지 레이아웃에 버튼 2개(start, stop)를 만든다.
2. Start버튼을 누르면 onclick 이벤트 리스터 에 연결된 함수실행
3. Start 버튼 누르면 시간 측정시작, 맞춰야하는 시간이 console에 출력
(시간의 범위는 랜덤으로 생성, 0s00ms~ 10s 99ms 까지 범위)
4. Stop 버튼을 누르면 시간 측정종료 , console.log로 시작 -종료 까지 시간 표시
(밀리초ms)까지 출력
5. 시작- 종료 까지의 시간이 정확히 start버튼을 눌렀을때 출력된 초 (XXsec XXms)를 맞추면 성공
6. Start를 눌렀을때 제시된 시간을 몇ㅁ번만에 맞추는지 count 하여 console에 시도한 회수 표기

document .write로 결과 출력시 모든 페이지 내용 (버튼들)사라지기때문에 
console에 표기
