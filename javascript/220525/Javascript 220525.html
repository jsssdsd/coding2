#객체의 개념
객체는 객체마다의 속성(property)과 기능(method)를 가지고 있다.
자바스크립트는 객체 기반의 언어
1. 기본 제공하는 객체
	1.1. 코어 객체 : Array, Date, String, Math
	1.2. DOM 객체 : html 태그 하나가 dom 객체
	1.3. 브라우저 객체(BOM) : 브라우저 관련 기능, 객체
	1.4. window 객체 : 윈도우 관련 기능 객체
2. 사용자가 직접 만드는 객체

# 코어 객체 생성 방법
코어 객체의 생성에는 키워드 new를 사용한다.
let x = new Date();
var y = new String();
